# Daily AI Workflow Optimization Templates

After six months of daily AI collaboration, I've learned that the secret isn't finding the perfect prompt—it's building a sustainable workflow that actually fits how you think and work.

## The Daily Startup Template

I start each session with context-setting that saves me hours of re-explanation:

```
Today I'm working on [specific project/goal]. 
My role is [your actual job function].
Current priority: [what you need to accomplish today].
Working style preference: [brief/detailed, creative/analytical, etc.]
Time constraint: [realistic timeframe]

First task: [specific request]
```

This 30-second setup prevents that frustrating dance where the AI gives generic advice while you need specific, contextual help.

## The Iteration Framework

Most people write one prompt and expect magic. Reality check: good outputs come from building on previous attempts. I use this progression:

**Round 1:** Basic request with minimal context
**Round 2:** "That's close, but adjust [specific element] because [reason]"  
**Round 3:** "Perfect. Now apply this same approach to [related task]"

The key insight? Each refinement teaches the AI your preferences, making subsequent requests exponentially better.

## Task-Specific Templates That Actually Work

**For analysis work:**
```
Analyze [data/situation] focusing on [specific angle].
Structure: Executive summary, 3 key findings, implications for [your domain].
Skip: General background I already know.
Include: Specific recommendations I can act on tomorrow.
```

**For creative projects:**
```
Generate [specific deliverable] for [exact audience].
Tone: [reference something concrete, like "NPR podcast style" not "engaging"]
Must include: [non-negotiable elements]
Avoid: [things that waste your time]
Give me 3 distinct approaches, not variations of the same idea.
```

## What Doesn't Work (Hard-Learned Lessons)

**Vague requests kill productivity.** "Help me write better" gets you a generic essay about writing. "Help me tighten this paragraph for a technical audience who needs to make a budget decision" gets you actionable edits.

**Over-prompting backfires.** I used to write 200-word prompts with every possible specification. Result? The AI focused on the wrong parts. Now I start simple and add complexity through conversation.

**Ignoring AI limitations wastes time.** AI excels at structure, patterns, and iteration. It struggles with nuanced judgment calls and real-time information. Design your workflow accordingly.

## The Honest Truth About Daily Use

Some days AI transforms my productivity. Other days I spend more time crafting prompts than just doing the work myself. The trick is recognizing which type of day you're having and adjusting expectations.

Build templates, but stay flexible. The goal isn't perfect prompts—it's a sustainable practice that makes your actual work better.
